# Phantom-Bookmarks

React - URL Bookmarking System

# Overview

This is bookmarks web app which lets you, bookmark your URLs, edit them and delete by your needs. With Pagination and clear all bookmarks feature.

# Tools used for development

Babel
Eslint
Material UI
Webpack
